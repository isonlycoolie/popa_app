### Design Annotations:

#### Typography:
- **App Title**: 18sp, 600 weight, #111827
- **Contact Names**: 16sp, 600 weight, #111827
- **Message Preview**: 16sp, 400 weight, #6B7280
- **Timestamps**: 13sp, 400 weight, #6B7280
- **Unread Count**: 11sp, 400 weight, #FFFFFF

#### Spacing:
- **Status Bar**: 24dp
- **App Bar**: 56dp
- **Search Bar**: 48dp height, 16dp horizontal padding
- **List Item**: 72dp height, 16dp horizontal padding
- **Bottom Navigation**: 80dp height
- **Item Spacing**: 1dp divider between items

#### Touch Targets:
- **List Items**: 72dp height (exceeds 48dp minimum)
- **Search Bar**: 48dp height (meets minimum)
- **Navigation Icons**: 48dp x 48dp (meets minimum)
- **Menu Button**: 48dp x 48dp (meets minimum)

#### Color Usage:
- **Background**: #FFFFFF
- **Surface**: #F7F7F8
- **Text Primary**: #111827
- **Text Secondary**: #6B7280
- **Accent**: #34C759 (unread indicators)
- **Border**: #E5E7EB (dividers)

#### Accessibility Features:
- **Screen Reader**: All elements have semantic labels
- **High Contrast**: Text meets WCAG AA standards
- **Touch Targets**: All interactive elements ≥48dp
- **Font Scaling**: Supports up to 200% zoom

#### Interactive Elements:
- **Search**: Tap to focus, keyboard appears
- **List Items**: Tap to open conversation
- **Navigation**: Tap to switch tabs
- **Menu**: Tap to open options

#### States:
- **Normal**: Default appearance
- **Selected**: Highlighted when tapped
- **Unread**: Bold text, unread count badge
- **Search**: Filtered results based on query

#### Responsive Behavior:
- **Portrait**: Full width list items
- **Landscape**: Maintains layout, adjusts spacing
- **Large Screens**: Maintains 375dp width, centers content
