### Design Annotations:

#### Typography:
- **Contact Name**: 18sp, 600 weight, #111827
- **Status Text**: 13sp, 400 weight, #34C759
- **Message Text**: 16sp, 400 weight, #111827
- **Timestamps**: 13sp, 400 weight, #6B7280
- **Date Separators**: 13sp, 400 weight, #6B7280

#### Spacing:
- **Status Bar**: 24dp
- **Chat Header**: 56dp
- **Message Area**: Flexible height, 16dp horizontal padding
- **Message Composer**: 80dp height, 16dp horizontal padding
- **Message Bubbles**: 8dp internal padding, 12dp between messages

#### Touch Targets:
- **Header Buttons**: 48dp x 48dp (meets minimum)
- **Attachment Button**: 48dp x 48dp (meets minimum)
- **Send Button**: 48dp x 48dp (meets minimum)
- **Message Bubbles**: 48dp minimum height

#### Color Usage:
- **Background**: #FFFFFF
- **Incoming Messages**: #FFFFFF (white bubbles)
- **Outgoing Messages**: #DCF8C6 (light green bubbles)
- **Text Primary**: #111827
- **Text Secondary**: #6B7280
- **Status Online**: #34C759
- **Message Status**: #34C759 (delivered/read)

#### Message Bubble Design:
- **Incoming**: White background, left-aligned, 16dp border radius
- **Outgoing**: Light green background, right-aligned, 20dp border radius
- **Max Width**: 78% of screen width
- **Padding**: 12dp horizontal, 8dp vertical
- **Shadow**: Subtle shadow for depth

#### Status Indicators:
- **Single Check (✓)**: Message sent
- **Double Check (✓✓)**: Message delivered
- **Blue Double Check**: Message read (future RCS)
- **Clock Icon**: Message sending
- **Exclamation**: Message failed

#### Interactive Elements:
- **Message Bubbles**: Tap to show timestamp
- **Attachment Button**: Tap to open attachment sheet
- **Send Button**: Tap to send message
- **Header Buttons**: Tap for call/video/options

#### Accessibility Features:
- **Screen Reader**: Messages announced with sender and content
- **High Contrast**: Clear bubble contrast
- **Touch Targets**: All interactive elements ≥48dp
- **Font Scaling**: Supports up to 200% zoom

#### States:
- **Normal**: Default message appearance
- **Sending**: Clock icon, disabled send button
- **Sent**: Single checkmark
- **Delivered**: Double checkmark
- **Failed**: Exclamation mark, retry option

#### Responsive Behavior:
- **Portrait**: Full width bubbles, proper spacing
- **Landscape**: Maintains layout, adjusts bubble width
- **Large Screens**: Maintains 375dp width, centers content

#### Message Types:
- **Text**: Standard text messages
- **Photos**: [📷] Photo indicator
- **Videos**: [🎥] Video indicator
- **Files**: [📄] File indicator
- **Audio**: [🎵] Audio indicator (future)
